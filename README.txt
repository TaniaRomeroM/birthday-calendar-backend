
/* BBDD */

 - Herramientas/puertos utilizados en el proyecto - 
Gestor de BBDD: HeidiSQL
Despliegue de MySQL: Docker
Puerto:33060
 
1º Arrancar el servidor MySQL, por ejemplo con docker(comando  ). 
2º Lanzar en el gestor de base de datos la BBDD con el script 'calendarbd.sql' DONDE ESTA, para crear la BBDD con los datos incluidos.



/* BACKEND */

 - Herramientas/puertos utilizados en el proyecto - 
IDE: IntelliJ IDEA
Puerto: 8080

1º Compilar el backend y ejecutarlo
2º Cambiar el archivo 'src/main/resources/static/application.properties' si se ejecuta en otro puerto.
3º Iniciar la aplicacion haciendo click en 'src/main/java/com.proyectocalendar.birthdaycalendar/BirthdaycalendarApplication' y pulsando Run.



/* FRONTEND */

 - Herramientas/puertos utilizados en el proyecto - 
Editor de codigo: Visual Studio Code
Puerto: 4200

1º Ejecutar el comando 'npm install'.
2º Ejecutar el servidor y abrir el navegador con el comando 'ng serve --open'.
